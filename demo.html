The baby tracker one family built for their newborn

Maya started as two exhausted new parents logging night feeds. It grew into a full privacy-first PWA: feeding timers, diaper photo analysis, a milk bank with live tank, sleep routines, stage-by-stage guidance — and a wellbeing check for mom.

▶ Launch interactive demo View source on GitHub

The demo runs entirely in your browser with synthetic data ("baby Emma"). Nothing is uploaded anywhere. Note: the product UI is in Spanish — built for a real Mexican family.

Everything a newborn's first months need

🤱Feeding timers

Left/right breast timers with pause, expressed milk and formula in ml — synced live between both parents' phones.

💩Smart diaper log

One-tap logging plus on-device photo analysis that suggests stool color and consistency, with pediatric guidance.

🥛Milk bank

Fridge and freezer inventory with a live tank gauge. Every bottle fed auto-deducts; pumping sessions add back.

⚡Guided pumping

Six methods including power pumping and Medela 2-phase, with phase alerts, records, charts and a production coach.

🌙Sleep, upgraded

Age-based wake windows with a traffic-light gauge, customizable bedtime routines and data-driven sleep tips.

🤖Local health analysis

Hydration, feeding, weight and condition trends vs. the baby's own baseline — computed on-device, never uploaded.

🗓️Stage guidance

Heads-up pop-ups as the baby enters each stage: colic season, teething, sleep regression — with relief tips.

🏆Daily challenges

Age-appropriate stimulation activities (tummy time timers!) with confetti, streaks and a medal shelf.

💚Mom's wellbeing

Scheduled Edinburgh Postnatal Depression Scale check-ins, private to mom's phone, with real help resources.

📅Appointments & meds

Medical appointments with native calendar alarms (.ics) and daily medication check-offs with reminders.

📄Pediatrician PDF

One tap exports charts and tables of any date range — feeds, diapers, sleep, bilirubin curves — for the doctor.

👶Up to 5 babies

Twins? Each baby gets a full separate record, one tap apart. Four color themes, dark mode included.

🔒 Privacy by design. No servers, no accounts database, no analytics. The family's data lives on their phones and syncs through their own private GitHub repository; the sync token ships AES-GCM-encrypted and only decrypts with the family's login. Photo analysis and health insights run 100% on-device.

Built with GitHub Pages · vanilla JS PWA · Chart.js · jsPDF — no build step, no backend.
