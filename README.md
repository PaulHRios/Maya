# 💗 Maya

Diario y registro de salud para nuestra bebé. Una app web pensada para usarse
desde el celular de mamá y papá, instalable en la pantalla de inicio.

## ¿Qué se puede registrar?

- 🍼 **Alimentación** — leche materna (con timer por pecho izquierdo/derecho),
  leche donante y fórmula, con mililitros y hora.
- 🌙 **Sueño** — timer de siestas o registro manual de horarios.
- 💧 **Pañales** — pipí, popó o ambos, con un solo toque.
- 🩺 **Condiciones médicas** — seguimiento de valores (por ejemplo bilirrubina)
  con gráficas, resultados pendientes e información de cuidados que la app
  busca automáticamente.
- 💉 **Intervenciones** — tomas de sangre, vacunas, estudios y consultas.
- 💊 **Medicamentos** — tratamientos y vitaminas.
- 📏 **Crecimiento** — peso, talla y perímetro cefálico.
- 📸 **Fotos** — recuerdos y fotos útiles para el pediatra.
- 📄 **Resumen en PDF** — reporte descargable con tablas y gráficas por día.

## ¿Cómo se usa?

1. Activa GitHub Pages en este repositorio (Settings → Pages → Deploy from a
   branch → rama principal, carpeta `/`).
2. Abre la página en el celular y usa **Compartir → Agregar a pantalla de
   inicio** (iPhone) o **Instalar app** (Android).
3. En **Más → Ajustes** configura la sincronización para que ambos celulares
   compartan los datos (se recomienda un repositorio **privado** aparte para
   guardarlos).

## Notas

- La app funciona sin internet; los datos se guardan en el dispositivo y se
  sincronizan cuando hay conexión.
- La información médica que muestra la app es orientativa y nunca sustituye
  la valoración del pediatra.
